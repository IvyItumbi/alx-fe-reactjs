# Form Handling in React

This project demonstrates form handling in React using both controlled components and Formik.

## Components

1.  **RegistrationForm**: Uses controlled components and manual state management.
2.  **formikForm**: Uses Formik library with Yup validation.

## Setup

1.  Navigate to `form-handling-react`.
2.  Run `npm install`.
3.  Run `npm run dev`.

## Dependencies

-   formik
-   yup
